1周年(2014年)から数えて、13年ごとに称号のスート(♠♡♢♣)がローテーションします。

12周年記念交流会 大会ルール
==============================
「BlackPoker 12周年記念交流会」における大会ルールです。

対戦レギュレーション
----------------------------------------
参加者はエントリー時に、対戦レギュレーションから希望するものを1つ選択します。

選択可能な対戦レギュレーションは次の通りです。

* **ライト + エントリー** 
* **スタンダード + パック** 
* **スタンダード + レアパック** 
* **プロ + パック** 
* **プロ + レアパック** 
* **プロ + ストラテジー** 
* **マスター + パック** 
* **マスター + レアパック** 
* **マスター + ストラテジー** 
* **マスター + エクストラ** 

レギュレーションの決定
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
対戦相手とのマッチング時、対戦レギュレーションは次の手順で決定します。

1. 対戦相手と希望する対戦レギュレーションが同じ場合、そのレギュレーションで対戦します。
2. 対戦相手と希望する対戦レギュレーションが異なる場合、両者で話し合って決定します。
3. 話し合いで決定できない場合、より左側のレギュレーション（ライト ＞ スタンダード ＞ プロ ＞ マスター の順）を希望していたプレイヤーに決定権があります。

対戦および勝点
----------------------------
大会は全6回戦で行われます。勝点（pt）は0ptから開始し、対戦結果に応じて次の通りに加算されます。

* **勝利**: 3pt
* **敗北**: 0pt
* **引き分け**: 1pt（両プレイヤーが獲得）
* **不戦勝（対戦相手不在）**: 3pt

※ BlackPokerは基本ルール上引き分けになることはありませんが、本大会では両プレイヤーの合意によって、試合を引き分けにすることができます。
