第9版の主な変更点（第8.2版 → 第9.0版）
=====================================

.. note::

   この文書は、第8.2版 PDF と第9.0版 PDF を比較して作成した変更履歴案です。
   ソースコードや YAML の差分ではなく、公開 PDF 上で確認できる変更をベースに整理しています。
   そのため、文言整理・章構成変更・表記統一も含みます。

概要
----

第9版は、単なる文言修正というより、次の3点が大きい版だと読み取れます。

1. ルール文書全体の整理
   付録構成の見直し、FAQ の追加、参照先の整理が行われています。
2. 魔法体系の再整理
   これまでの「速攻魔法 / 通常魔法」という見せ方から、
   「基礎魔法 / 中級魔法 / 上級魔法」という難易度・習熟度寄りの整理へ移っています。
3. エクストラの実戦調整
   特に切札まわりで、名称変更、新規アクション追加、条件緩和、クイック化が複数あります。
   エクストラを遊んでいるプレイヤーは重点確認が必要です。

最重要変更（プレイヤー向け）
----------------------------

1. 魔法分類の整理変更
~~~~~~~~~~~~~~~~~~~~~~

変更前
   7.3 のアクションリストは「速攻魔法」「通常魔法」で整理されていました。

変更後
   第9版では「基礎魔法」「中級魔法」「上級魔法」に再編されています。

読み替えの目安
   * 基礎魔法
     アップ / ダウン / ツイスト / カウンター
   * 中級魔法
     防壁破壊 / 投擲 / 死の槍 / 防壁補充 / リアニメイト / ハンデス /
     キル / 再会 / 停戦 / 対象変更 / サーチ / リバース / 帰還
   * 上級魔法
     剣の雨 / フォース / 徴募 / 奇襲 / B・J / R・S・F

プレイへの影響
   ここは主に分類名の変更です。
   ただし、第9版では「そのアクションが実際にいつ打てるか」は、
   種別名よりも ``@直接-通常-メイン`` や ``@直接-即時-クイック`` の表記を
   直接読む運用へ寄っています。

2. 切札 ♡3 は「激励」から「誘惑」に変更
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

変更前
   * 切札名: 激励
   * 兵士起因アクション: ``激励``
   * 効果: 対象兵士のサイズをこのターン中 X 加算する。

変更後
   * 切札名: 誘惑
   * 兵士起因アクション: ``誘惑``
   * 効果: 対象の兵士をオーナーの手札に戻す。
   * さらに ``クイック兵士召喚 ♡`` が追加されました。

プレイへの影響
   これはかなり大きい変更です。
   旧版の♡3は「打点補助・戦闘補助」寄りでしたが、第9版の♡3は
   「盤面の兵士を剥がす」「相手兵士をテンポ損させる」方向に役割が変わっています。
   また、同時にクイック召喚手段も得ているため、
   ♡系の切札としては一時強化よりも、差し戻しと展開補助を担う札として読むべきです。

3. 切札 ♢5 は「疲弊」から「虚脱」に変更
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

変更前
   * アクション名: ``疲弊``
   * 種別: 誘発魔法
   * 内容: エンドアクションをリクエストした時に誘発し、
     ``喪失`` を裏返すか、そうしなければ 2 ダメージを受ける。

変更後
   * アクション名: ``虚脱``
   * 種別: 魔法
   * 内容: ``♢ A～K`` を使い、ドライブ状態のキャラクター 1 体を墓地に移す。

プレイへの影響
   旧版では「維持コスト / デメリット管理」寄りだったのに対し、
   第9版では明確な能動除去へ変わっています。
   名前だけでなく機能も別物と考えた方が安全です。
   旧来の ``疲弊`` 前提でプレイすると誤解しやすい箇所です。

4. 切札 ♣9 要塞に「反撃」が追加
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

変更前
   切札 ``要塞`` 自体の能力のみでした。

変更後
   ``反撃`` という誘発アクションが追加されました。
   条件は「自分の表切札に要塞がある」かつ
   「自分の防壁がダメージ判定でブロッカーとして墓地に移った時」で、
   効果は「アタッカーのオーナーに 2 ダメージ」です。

プレイへの影響
   要塞デッキの防壁運用に、明確な報復性能が追加されました。
   防壁が割られて終わりではなく、相手の攻撃側にも確定ダメージが返るため、
   防壁を挟んだ攻防の価値が上がっています。

5. 切札 ♢A「防壁追加」が使いやすく調整
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

変更前
   * 1ターン1回
   * 相手の防壁数が自分以上である場合にのみ発動可能
   * ライフの一番上から 1 枚を防壁として配置

変更後
   * 1ターン1回なのは同じ
   * 相手との防壁数比較条件が消えています
   * ライフの一番上から 1 枚を防壁として配置
   * さらに ``クイック防壁設置`` が追加
     （コスト ``DL`` で手札から防壁をクイック配置）

プレイへの影響
   第9版では、防壁追加がかなり素直な防御札になりました。
   盤面条件を待たずに使えるため腐りにくく、
   さらにクイック防壁設置により、防御の即応性まで上がっています。

6. 切札 ♢4「相殺」は発動条件が緩和
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

変更前
   対象リクエストのキーカード数字合計が ``5 以上`` の場合にのみ成立しました。

変更後
   数字合計が 5 以上という条件が消え、
   対象リクエストのキーカード数字合計ぶんダメージを受けたうえで、
   そのリクエストを取り除けるようになっています。

プレイへの影響
   低い数字のリクエストにも触れるようになったため、
   使える場面が増えました。
   反面、自傷コストは常に発生するので、ライフ管理はより重要です。

7. 切札 ♠5「蘇生」は対象範囲が拡大
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

変更前
   墓地の ``♠2～10`` を 1 枚選び、兵士として場に出す。

変更後
   墓地の ``♠のカード`` を 1 枚選び、兵士として場に出す。

プレイへの影響
   数札限定から、より広い♠カードへ対象が広がっています。
   デッキ構築と墓地利用の幅が増えたと見てよいです。

8. 切札 ♢8「修繕」は対象と素材条件が緩和
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

変更前
   * 対象: 自分の場にいる ``♢2～10`` を含んだ兵士 1 体
   * 墓地から選ぶカード: ``♢J～K``

変更後
   * 対象: 自分の場にいる兵士 1 体
   * 墓地から選ぶカード: 対象兵士と ``同じスート`` のカード 1 枚

プレイへの影響
   旧版よりかなり柔らかい条件になっています。
   ♦専用ギミック色が薄れ、同スート連携の装備支援札として扱いやすくなりました。

9. 切札 ♣7「手札補充」は強化
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

変更前
   * 手札 2 枚以下なら 2 ドロー
   * 手札 3 枚なら 1 ドロー

変更後
   * 条件文は同じ誘発条件ですが、効果は ``2 枚ドロー`` に整理されています。

プレイへの影響
   旧版より期待値が上がっています。
   手札 3 枚時の弱い補充ではなく、明確なリソース回復札として見てよさそうです。

10. 切札 ♣8「リセット」はクイック化・簡略化
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

変更前
   * 種別: 通常魔法
   * タイミング: メイン
   * キーカード条件: ``♣A～K``

変更後
   * 種別: 魔法
   * ``@直接-即時-クイック``
   * キーカード条件の記載が見当たりません
   * 効果本体は同じ

プレイへの影響
   盤面更地の切り返し札として、かなり危険度が上がっています。
   特に「メインでしか打てない全体リセット」から
   「クイックで飛んでくる全体リセット」へ寄った点は、対戦感覚に大きく影響します。

11. 切札 ♡5「追撃」は使用後に閉じるよう変更
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

変更前
   * 兵士全体をチャージ
   * アタックアクションをリクエスト

変更後
   上記 2 つに加え、
   * ``自分の切札にある ♡5 を裏向きにする``
     が追加されました。

プレイへの影響
   追撃の連発抑制、あるいは再起動までのテンポ調整が入ったと読めます。
   旧版より、使い切り寄りの一撃札になっています。

12. 「魔王の囁き」の条件変更
~~~~~~~~~~~~~~~~~~~~~~~~~~~

変更前
   ``自分の表切札に「魔王」がある`` ときに誘発。

変更後
   ``自分の場に「魔王」がいる`` ときに誘発。

プレイへの影響
   これは見逃しやすいですが重要です。
   第9版では、単に切札を表にしているだけではなく、
   実際に魔王ユニットが場にいることが要求されています。

13. 新規のクイック展開手段が増加
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

第9版では、次の新規アクションが追加されています。

* ``クイック兵士召喚 ♠``
* ``クイック兵士召喚 ♡``
* ``クイック兵士召喚 ♢``
* ``クイック兵士召喚 ♣``
* ``クイック防壁設置``
* ``クイック威圧``

プレイへの影響
   エクストラは第8.2版よりも、相手ターン中の展開・応酬・差し込みが強くなっています。
   第9版を遊ぶ場合、相手の表切札を見た時点で、
   「そこからクイックで何が飛んでくるか」を旧版以上に意識する必要があります。

文書構成・参照性の変更
----------------------

1. 付録構成の拡張
~~~~~~~~~~~~~~~~~~

変更前
   第10章の付録は、主に ``アクションリスト`` と ``ガイドライン`` で構成されていました。

変更後
   第10章は次のように整理されています。

   * PDF版ルール
   * アクションリスト
   * エクストラリスト
   * ガイドライン

影響
   参照先が明確になり、特にエクストラ関係を追いやすくなっています。

2. FAQ 的な小見出しが増加
~~~~~~~~~~~~~~~~~~~~~~~~~~

第9版では、共通ルールの中に ``よくある質問`` が複数追加されています。

* 切札の持つ項目
* 誘発チェック
* 共通ルール: その他

影響
   実戦中の疑問点に対して、本文中から直接たどりやすくなっています。

3. 見出し・用語の整理
~~~~~~~~~~~~~~~~~~~~~~

確認できる範囲では、次のような整理があります。

* ``能力の適応順`` → ``能力の適用順``
* 基本ルール中の一部補足節の統廃合
* 付録やエクストラリストへの導線整理

影響
   このあたりはルールそのものの変更というより、
   読みやすさ・参照性の改善として読むのがよさそうです。

プレイヤー向け確認チェックリスト
--------------------------------

エクストラを遊ぶプレイヤーは、最低でも次を確認することを推奨します。

1. ♡3 の運用を旧 ``激励`` ベースの感覚から切り替えたか
2. ♢5 を旧 ``疲弊`` の維持札ではなく除去札として認識しているか
3. ♣9 ``要塞`` に ``反撃`` が増えたことを前提に攻撃判断しているか
4. 防壁追加 / リセット / 相殺 / 手札補充 の挙動変化を反映したか
5. クイック兵士召喚群とクイック防壁設置をケアした立ち回りになっているか
6. 魔法の分類名を、旧 ``速攻魔法 / 通常魔法`` のまま覚えていないか
