# RSTドキュメント記述スタイルガイド

このスキルは、BlackPokerプロジェクト内の `.rst` ファイルを編集または作成する際に、記述スタイルや用語の一貫性を維持・チェックするためのものです。

## 表記・用語の統一ルール

### 1. 指示語としての「以下」の制限
- **原則**: 後続の文章、表、画像、手順などを指し示す場合は「以下」ではなく **「次」** を使用します。
  - **非推奨（例）**: 「以下の手順に従って選択してください」「以下の表をご覧ください」「以下に、段階ごとの流れを示します」
  - **推奨（例）**: 「次の手順に従って選択してください」「次の表をご覧ください」「次に、段階ごとの流れを示します」
- **例外**: 数値や数量の範囲を示す不等号としての「以下」は、変更せずにそのまま使用してください。
  - **例外例**: 「0以下」「ライフ2以下」「10枚以下」

## チェック手順
`.rst` ファイルを作成または修正する際は、必ず本スキルに定義されたルールと照らし合わせ、不適切な「以下」の表現が含まれていないかをチェックしてください。
