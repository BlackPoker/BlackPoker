通常フォーマット¶

この章では、共通ルールを踏まえた通常フォーマットのルールを説明します。

共通ルールでは、具体的なアクションの効果は説明していませんでした。

BlackPokerでは共通ルールとフォーマットに応じたアクションリストをあわせることで、 対戦することが出来ます。

前提となるルールは、共通ルールを参照して下さい。
